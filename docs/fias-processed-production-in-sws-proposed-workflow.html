Chapter 1 FIAS processed production in SWS: proposed workflow

Figure 1.1 reprensents the essential points of the overall workflow for processed production data.

Figure 1.1: Workflow for processed production data.

The first part of data collection and data entry is entirely performed by FIAS unit; the second part of data imputation and validation uses the ‘ad-hoc’ shiny application; the third part is an R-plugin to be run on the SWS. The following chapter describes the process with emphasis on the data imputation and validation part.
